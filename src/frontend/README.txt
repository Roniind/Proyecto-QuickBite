# Frontend de QuickBite

Este directorio contiene las vistas básicas (HTML y CSS) que representan el prototipo visual de la plataforma **QuickBite**.

## Contenido

- `index.html`: Página principal de bienvenida.
- `login.html`: Formulario simple de inicio de sesión.
- `restaurantes.html`: Simulación de listado de restaurantes.
- `estilos.css`: Archivo de estilos básico (opcionalmente con Bootstrap).

## Relación con la nube

- Este frontend se conecta con una **API ubicada en un entorno de nube privada**.
- Toda la comunicación (como obtener lista de restaurantes o enviar un pedido) se haría mediante solicitudes HTTP a endpoints seguros en la nube.
- No contiene lógica sensible, sólo la presentación visual del sistema.

## Nota

Este frontend es parte de un **prototipo funcional académico** para mostrar cómo se estructuraría una solución basada en Cloud Computing.
